# <a name="_toc211263906"></a>Цель работы
Целью лабораторной работы является анализ и сравнение производительности (скорости вычислений) на графических процессорах nVidia (GPU) с последовательными и другими параллельными вычислительными системами (OpenMP, MPI).

# <a name="_toc211263907"></a>Задачи
Вариант индивидуального задания определяется по номеру в списке (№ варианта = № в списке группы).

Выполнить программные реализации алгоритмов согласно номеру варианта. 

1. Реализовать последовательны алгоритм.
1. Реализовать параллельный алгоритм с использованием технологии OpenMP.
1. Реализовать параллельный алгоритм с использованием MPI.
1. Реализовать параллельный алгоритм для вычислений для GPU nVidia (CUDA).
1. Провести отдельный вычислительный эксперимент для задач 1 - 4:
   1. Определить время фактического выполнения программы
   1. Построить графики **зависимостей времени** выполнения **от размера входных данных**.
1. Провести сравнительный анализ результатов вычислительных экспериментов для задач 1 – 4.
1. Сделать отчет. Имя файла *«ФамилияИмяГруппаЛаб№»*.
